# 🏨 Restaurant and Hotel Job Management System

A comprehensive web-based application designed to streamline human resource operations in the hospitality industry. This system automates hiring processes, manages staff shifts, and tracks daily task assignments to improve operational efficiency.

---

###  **Live Demo:** https://rh-jobs-ovi.kesug.com

---

## Key Features

* **Automated Hiring:** Streamlines the recruitment process for restaurant and hotel staff.
* **Shift Scheduling:** Efficiently manages and assigns staff shifts to ensure 24/7 coverage.
* **Task Management:** Assigns and tracks daily tasks for different departments.
* **Performance Tracking:** Handles complex staff performance data and scheduling logs.
* **Role-Based Access:** Secure **Admin** and **Staff** authentication using SQL.

## Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap.
* **Backend:** PHP (Server-side logic).
* **Database:** MySQL (Relational database for secure data handling).
* **Deployment:** Hosted on InfinityFree / Kesug.
